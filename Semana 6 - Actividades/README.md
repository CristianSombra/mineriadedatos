## Descripción

En este repositorio se desarrollan las actividades correspondientes a la clase 6 de Minería de Datos, enfocadas en técnicas de clustering y segmentación de datos utilizando algoritmos de agrupamiento aplicados mediante Python y herramientas de Machine Learning.

## Contenidos

* Introducción al clustering
* Segmentación de datos
* Algoritmo K-Means
* Funcionamiento de centroides
* Método del codo
* Selección de cantidad de clusters
* Algoritmo DBSCAN
* Agrupamiento basado en densidad
* Parámetros `eps` y `min_samples`
* Detección de ruido y outliers
* Similitudes entre K-Means y DBSCAN
* Diferencias entre K-Means y DBSCAN
* Comparación entre algoritmos
* Interpretación de resultados
* Aplicaciones prácticas del clustering
* Actividad integradora y presentación en video
