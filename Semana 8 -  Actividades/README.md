## Descripción

En este repositorio se desarrollan las actividades correspondientes a la clase 8 de Minería de Datos, enfocadas en aprendizaje supervisado mediante técnicas de clasificación y regresión aplicadas con Python y herramientas de Machine Learning.

## Contenidos

* Introducción al aprendizaje supervisado
* Diferencias entre clasificación y regresión
* Regresión Logística
* Función sigmoide
* Clasificación binaria
* Regresión Lineal Múltiple
* Variables predictoras y variable objetivo
* División de datos en entrenamiento y prueba
* Escalado y preprocesamiento de datos
* Matriz de confusión
* Accuracy
* Precision
* Recall
* F1-score
* Coeficientes de regresión
* R²
* MSE y MAE
* Interpretación de métricas
* Selección y comparación de modelos
* Aplicación práctica sobre datasets Titanic y California Housing
* Ejercicio integrador
* Reflexión final y análisis de resultados
