# Minería de Datos

Descripción: Repositorio con los trabajos prácticos y actividades de la materia Minería de Datos.

Organización: Cada carpeta corresponde a una clase e incluye los ejercicios desarrollados en Jupyter Notebook.

## Contenido actual

- Clase 1: conceptos básicos, KDD, CRISP-DM y análisis inicial de datos
- Clase 2: tipos de datos, limpieza, transformación y análisis exploratorio (EDA), incluyendo detección de outliers, codificación de variables categóricas y actividad integradora con el dataset Titanic
- Clase 3: análisis exploratorio de datos (EDA), análisis univariado y multivariado, visualización de datos, correlación, reducción de dimensionalidad con PCA y clustering (K-means) con actividad integradora
