## Descripción

En este repositorio se desarrollan los ejercicios correspondientes a la clase 4 de Minería de Datos, enfocados en la visualización de datos, selección de gráficos adecuados y análisis visual mediante herramientas de Python como Matplotlib y Seaborn.

## Contenidos

* Importancia de la visualización de datos
* Tipos de gráficos y cuándo utilizarlos
* Histogramas
* Boxplots
* Gráficos de barras
* Scatter plots
* Comparación entre Matplotlib y Seaborn
* Pairplot
* Heatmap de correlaciones
* Violin Plot
* Interpretación de patrones visuales
* Análisis exploratorio mediante gráficos
* Aplicación práctica en dataset de ventas
