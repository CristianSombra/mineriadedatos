# Descripción
En este repositorio se desarrollan los ejercicios correspondientes a la clase 2 de Minería de Datos.

## Contenidos
- Tipos de datos y su clasificación  
- Limpieza de datos (valores faltantes, inconsistencias y outliers)  
- Transformación de datos categóricos (codificación ordinal y one-hot)  
- Análisis exploratorio de datos (EDA)  
- Estadísticas descriptivas y correlación  
- Detección de outliers con IQR
