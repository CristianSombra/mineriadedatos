## Descripción

En este repositorio se desarrollan los ejercicios correspondientes a la clase 3 de Minería de Datos, enfocados en el análisis exploratorio de datos (EDA), reducción de dimensionalidad y técnicas de agrupamiento.

## Contenidos

* Análisis exploratorio de datos (EDA)
* Análisis univariado (distribución, media, mediana, outliers)
* Análisis multivariado (correlación y relaciones entre variables)
* Visualización de datos (histogramas, boxplots y scatter plots)
* Matriz de correlación
* Reducción de dimensionalidad con PCA
* Interpretación de componentes principales
* Clustering (K-means)
* Segmentación de datos
* Comparación de resultados con y sin PCA
* Interpretación de clusters
* Integración de técnicas en un caso práctico
