## Descripción

En este repositorio se desarrollan los ejercicios correspondientes a la clase 5 de Minería de Datos, enfocados en el preprocesamiento, limpieza y transformación de datos utilizando herramientas de Python y bibliotecas de Machine Learning.

## Contenidos

* Importancia del preprocesamiento de datos
* Limpieza de datos
* Manejo de valores nulos
* Tipos de datos faltantes (MCAR, MAR y MNAR)
* Detección de outliers mediante IQR
* Winsorización
* Transformación logarítmica
* Normalización con Min-Max Scaling
* Estandarización con Z-score Scaling
* Uso de RobustScaler
* One-Hot Encoding
* Label Encoding
* Comparación entre variables originales y transformadas
* Construcción de pipelines con Scikit-learn
* Aplicación práctica sobre datasets de ejemplo
